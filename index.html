Teams MS

View Website Watch Video

Teams MS is a team management tool that makes project building process much more smoother, with built in features like directory hoasting, productivity monitoring and task management and much more on personal and project level. It is powered with access control that helps team owner have some certian permissions that an Admin and a developer (Roles) doesn't, Where an Owner can also promote there members to grant access to some aditional permissions in a project.

Features:
  1. Access Control (Owner, Admin, Developer )
  2. Task Assignment
  3. File and directory sharing as well as file search
  4. Project Logs as well as personalized notifications
  5. Create Personal Notes - Marking as Done & Delete
  6. Public and Private Projects

Technologies Used:
  1. Angular
  2. Node.js / Express.js
  3. Google Analytics
  4. MongoDB Atlas
  5. Firebase
